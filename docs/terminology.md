# Terminology

Use these terms consistently in this repo:

- `Azure AI Foundry`
  - preferred current term for Microsoft's Azure AI platform direction
- `Azure AI Search`
  - use instead of older search naming where possible
- `Managed Identity`
  - preferred identity pattern for Azure-hosted workloads
- `Azure API Management`
  - preferred term for the API gateway and policy layer
- `RAG`
  - retrieval-augmented generation
- `Golden Path`
  - the currently recommended, validated execution path

Avoid these patterns:

- mixing old and new product names without explanation
- calling the full repo production-ready when only one path is validated
- using `enterprise-grade` as a marketing label without linked evidence
