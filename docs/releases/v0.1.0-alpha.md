# v0.1.0-alpha

## Summary

`v0.1.0-alpha` is the first structured prerelease of this repository.

It means the repo now has:

- explicit governance and release boundaries
- smoke validation
- multiple runnable module baselines
- policy and RBAC examples

## Runnable Modules in This Alpha

- `09-end-to-end-reference-implementation`
- `05-rag-with-azure-ai-search`
- `07-secure-reference-architecture`
- `08-observability-and-evaluation`

## What This Alpha Does Not Claim

- full production readiness
- complete policy enforcement
- complete private networking
- mature live observability or evaluation operations
- complete promotion of all modules

## Recommended Reader Path

1. start with `09-end-to-end-reference-implementation`
2. review `KNOWN-LIMITATIONS.md`
3. review `VALIDATION-MATRIX.md`
4. use modules 05, 07, and 08 as secondary runnable baselines

## Release Positioning

This alpha is intended to communicate progress honestly:

- more than a draft repo
- less than a production-ready reference standard
