# Multi-Agent System Orchestration

This lab module explores advanced multi-agent orchestration patterns using Semantic Kernel and AutoGen.

### Learning Objectives:
- Master hierarchical and group chat orchestration patterns
- Implement agent state persistence and session management
- Design multi-agent workflows for complex research and analysis
- Implement error handling and graceful degradation in multi-agent systems
- Optimize agent communication and token usage

### Lab Contents:
- **Lab 1**: Building a Hierarchical Supervisor pattern
- **Lab 2**: Group Chat Orchestration with AutoGen
- **Lab 3**: Implementing Task-Specific Agents (Researcher, Writer, Reviewer)
- **Lab 4**: Persistent Multi-Agent Sessions with Cosmos DB
- **Lab 5**: Monitoring Multi-Agent interactions with LangSmith/Azure Monitor
