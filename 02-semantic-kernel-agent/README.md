# Semantic Kernel Agent Architecture

This lab module focuses on building enterprise-grade agentic AI solutions using the Semantic Kernel SDK. 

### Learning Objectives:
- Master Semantic Kernel architecture and its core components
- Develop and integrate custom plugins (native and semantic)
- Orchestrate complex tasks using Planners
- Implement state and memory management in agentic workflows
- Design for scale and reliability in agentic systems

### Lab Contents:
- **Lab 1**: Initializing Semantic Kernel and basic prompt orchestration
- **Lab 2**: Creating specialized plugins for business logic
- **Lab 3**: Autonomous goal-seeking with Handlebars and Function Choice Call planners
- **Lab 4**: Persistent state management with Cosmos DB
